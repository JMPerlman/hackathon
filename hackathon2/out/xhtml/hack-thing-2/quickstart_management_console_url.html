Get Your Management Console URL

  1. Go back to the HP Helion Public CLoud management console.
  2. From the Compute menu, select Instances.
    The cluster will be running on a single virtual machine in your account that will have the name cluster1-core
  3. Write down the public facing IP (15.X.X.X) under the IP Address Column
  4. Your URL will be https:://<IP>.xio.io. For example:
    Use this URL in your web browser to navigate to the web-based ALS management console.
