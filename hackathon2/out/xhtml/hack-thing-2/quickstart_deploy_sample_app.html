Deploy an Application from the Sample Application Gallery

  1. Scroll down and click the Deploy from the Sample Applications button.
  2. Click Deploy App in one of the sample apps (we recommend starting with a hello world app). For example:
  3. Fill out the application deployment form:
    1. Enter a name in the App Name text box.
    2. Select a space from the Space drop-down list box.
    3. Select domain in the 15.X.X.X address range.
    4. Check the Start application after deployment check box.
    5. Click Deploy Application.
    The View App button will be enabled before the app is ready. Usually it takes a few minutes for the application to be fully deployed. When the app is fully deployed, the log stream will show output similar to the following: 
    http://helion-helloworld-java-edmwx.15.X.X.X.xip.io/ deployed
All done.
  4. Click View App.
Congratulations! You have now deployed your first application!
